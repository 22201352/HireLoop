# HireLoop

An AI-driven hiring platform streamlining recruitment through smart matching, role-based access, and admin-approved workflows.

Built with Next.js 14, MongoDB (native driver), and Bootstrap.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Frontend:** Next.js 14, Bootstrap
- **Backend:** Next.js API routes
- **Database:** MongoDB (native driver)

## Features

- Role-based authentication (Candidate / Recruiter / Admin)
- Admin approval flow for recruiters
- Job posting and job listing with filters
- Candidate and recruiter dashboards
